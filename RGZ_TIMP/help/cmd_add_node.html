Справка: Добавление нового узла

Добавление узла (вершины) — это первичная операция конструирования схемы.

Порядок действий:

  1. Нажмите правую кнопку мыши (ПКМ) на любой пустой области полотна (по белому фону). Важно: кликать надо не на существующий узел, а на фон.
  2. Появится всплывающее меню. Выберите пункт "Добавить".
  3. В месте, где вы кликнули, будет создан новый узел с базовым состоянием. По умолчанию он получит внутренний C# код.

Для того чтобы посмотреть или изменить свойства добавленного узла, сделайте по нему двойной клик левой кнопкой мыши. Если вы измените код некорректно, система выдаст ошибку "Некорректный код узла".
