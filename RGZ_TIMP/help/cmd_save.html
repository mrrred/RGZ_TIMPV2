Справка: Команда "Сохранить"

Функция для фиксации текущего достижения пользователя в файл на жёсткий диск.

Формат данных: Проект сохраняется в формате XML (Extensible Markup Language). Этот формат удобен не только для чтения программой, но и для просмотра людьми через любой текстовый редактор. В файл записываются координаты отрисовки, код C# каждого узла и параметры дуг.

Как сохранить проект:

  • Нажмите "Сохранить".
  • В диалоговом окне введите имя (можно использовать стандартное, предлагаемое программой: GraphProject.xml) и выберите папку.
  • Нажмите "Сохранить".
