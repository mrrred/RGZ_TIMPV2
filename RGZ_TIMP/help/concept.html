Общая справка по приложению "Графопостроитель"

1. Введение (общее описание)

Назначение программы: Программа "Графопостроитель" предназначена для визуального построения графов, добавления узлов с программным кодом (на языке C#) и ориентированных рёбер со встроенными условиями (предикатами) и задержками времени.

Область применения: Данный программный продукт может использоваться для моделирования логики вычислений, образовательных целей в области теории графов, а также для визуальной демонстрации выполнения пошагового алгоритма. Приложение подходит для использования студентами, преподавателями и инженерами-проектировщиками.

Основные функции:

  • Создание, загрузка и сохранение (в формате XML) проектов направленных графов;
  • Редактирование узлов с вводом C# кода;
  • Задание логических условий (предикатов) и временных задержек на рёбрах графа;
  • Пошаговая визуальная анимация выполнения построенной модели с расчетом состояния программы.

Версия программы: 1.0 (2024)
Разработчик: Студент (в рамках курсовой/расчетно-графической работы).

2. Системные требования и установка

  • ОС: Windows 10 или новее.
  • Процессор: 1 ГГц или выше.
  • ОЗУ: Минимум 2 ГБ.
  • Свободное место: Не менее 100 МБ.
  • Платформа: Установленный .NET 10.0 Runtime.

Установка: Программа портативная и не требует установки. Для запуска достаточно распаковать архив с программой в любую удобную папку и запустить исполняемый файл RGZ_TIMP.exe.

3. Обзор интерфейса

Рабочее окно программы условно делится на три основные зоны:

  • Строка меню (сверху): Содержит пункты для работы с файлами проекта (Создать, Открыть, Сохранить), а также кнопки управления процессом анимации (Анимация, Выполнить, Остановить).
  • Визуальное полотно (содержимое по центру): Главная рабочая область. Здесь происходит отрисовка узлов и связей между ними. Узлы отображаются кругами с номерами, а рёбра — стрелками.
  • Строка состояния (снизу): Отображает текущий режим работы приложения (например, "Режим редактирования" или "Режим анимации").

Подсказка: Для получения дополнительной справки по конкретным элементам управления нажмите клавишу F1, когда курсор мыши наведен на элемент.

4. Основные возможности (процедурная часть)

  • Как начать новый проект? Выберите в меню Команды → Создать. Текущее полотно будет очищено.
  • Как открыть/сохранить файл? Выберите в меню Команды → Открыть / Сохранить. Данные сохраняются в понятном формате XML.
  • Как добавить узел? Кликните правой кнопкой мыши по пустому месту на полотне и нажмите "Добавить".
  • Как соединить узлы? Зажмите клавишу Ctrl, кликните по чёрному маркеру узла-родителя и протяните мышь до другого узла.
  • Как начать анимацию? Нажмите в меню кнопку Выполнить. Для настройки длительности шагов используйте пункт Анимация.

5. Дополнительные сведения

Горячие клавиши:

  • Ctrl+Z — Отменить последнее действие (в режиме редактирования).
  • Ctrl+Shift+Z — Вернуть отмененное действие.
  • F1 — Контекстная справочная система.
