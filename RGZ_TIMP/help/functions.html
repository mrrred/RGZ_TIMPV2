Описание функций

Главное меню

  • Создать — создаёт новый пустой граф.
  • Открыть — открывает сохранённый граф из XML-файла.
  • Сохранить — сохраняет текущий граф в XML-файл.
  • Анимация — настройка параметров анимации (длительность задержки между узлами для визуализации).
  • Выполнить — запускает анимацию выполнения графа.
  • Остановить — останавливает текущую анимацию и возвращает узлы в исходное состояние.
  • ? (Справка) — открывает это окно с общей справкой.

Элементы графа

  • Узел — точка на полотне, в которой можно задать C# код (выполняется во время анимации).
  • Ребро — связь между узлами, содержит логический предикат (условие перехода) и задержку (дополнительное время перехода).
