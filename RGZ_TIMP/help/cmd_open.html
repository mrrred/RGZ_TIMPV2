Справка: Команда "Открыть"

Эта команда используется для загрузки ранее сохранённой структуры графа с жёсткого диска компьютера.

Поддерживаемый тип файлов: Программа работает с файлами, имеющими расширение *.xml. Проект сохраняется в формате сериализованных данных, где задокументированы позиции узлов, их код, параметры рёбер (предикаты и задержки).

Как это работает:

  1. При нажатии появляется стандартное диалоговое окно Windows для выбора файла.
  2. Вы находите файл в нужной директории и нажимаете "Открыть".
  3. Приложение очищает текущую сцену (то, что было на экране, будет заменено) и отрисует узлы в соответствии с информацией в файле.
