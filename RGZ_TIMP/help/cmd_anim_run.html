Справка: Команда "Выполнить" (Анимация графа)

Кнопка "Выполнить" запускает процесс обхода графа. Т.е. мы визуально исполняем алгоритм, заложенный в структуру узлов и рёбер.

Как подробно выполняется анимация:

  1. Программа переходит в режим "Анимация". Блокируются возможности перемещать, добавлять или удалять узлы.
  2. Вычисление узлов: При активации узла (он подсвечивается определенным цветом), программа, через специальный интерпретатор C# кода (Roslyn C# Scripting), компилирует и выполняет "в реальном времени" код, записанный в данном узле.
  3. Получение результата: Узел возвращает целое число (псевдослучайное или жестко заданное), которое используется как результат узла.
  4. Оценка путей (Предикатов): Система анализирует все исходящие дуги от узла. Для каждой дуги проверяется её Предикат. Переход по ребру осуществляется только если предикат дуги совпадает с числом, которое только что выдал узел. Если нет совпадающих путей — алгоритм прекращается в этом узле (тупик).
  5. Ожидание задержки: Перед тем, как перейти к следующему узлу, запускается системный таймер. Длительность ожидания складывается из двух величин: "Длительность (базовая из настроек анимации)" + "Собственная задержка (Delay) конкретного ребра".
  6. Параллелизм: Если несколько рёбер имеют подходящий предикат, граф расщепляется и движение продолжается параллельно по всем ветвям.

Анимация закончится, когда все активные "маркеры" попадут в узлы, из которых нет выхода, или их результат не будет соответствовать ни одному выходящему предикату.
